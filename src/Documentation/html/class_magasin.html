Detailed Description

template<template< typename K, typename V > class TableAssociative>
class Magasin< TableAssociative >

Patron de classe implémentant un magasin de Produits.

Le magasin dispose d'un catalogue de produits ; Il associe à chacun un prix unitaire et un stock ; Le paramètre template TableAssociative<K,V> définit la structure de stockage des informations ; il doit être conforme à la spécification de AListe
